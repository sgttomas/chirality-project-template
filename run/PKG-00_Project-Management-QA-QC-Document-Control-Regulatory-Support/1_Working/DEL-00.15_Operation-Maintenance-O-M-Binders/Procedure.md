## 1. Purpose

### 1.1 Objective

This procedure defines the step-by-step process for compiling, reviewing, and delivering Operation & Maintenance (O&M) Binders for the Puget Sound Optimization Project (PSO).

### 1.2 Scope

This procedure covers:
- O&M binder planning and setup
- Vendor manual collection and tracking
- Content compilation and organization
- Binder review and quality control
- Binder submission and delivery

## 4. Verification

### 4.1 Quality Control Checklist

| Check Item | Verification Method | Pass Criteria |
|------------|---------------------|---------------|
| All equipment covered | Compare to equipment list | 100% coverage |
| All sections populated | Review each section | No missing sections |
| Vendor manuals included | Visual check | Manuals present |
| Drawings current | Check revision levels | IFC/As-Built revisions |
| Spare parts lists complete | Review against vendors | Lists present |
| PDF navigation works | Test bookmarks | All links functional |
| Binders registered in MDR | Check MDR | Entry exists |

### 4.2 Review Requirements

| Milestone | Reviewer | Approval |
|-----------|----------|----------|
| Draft Binder | Discipline Leads | Content validated |
| Internal Review | Engineering Lead | Internal approval |
| TM Submission | TM Operations/Maintenance | TM acceptance |

### 4.3 Issue Resolution

| Issue Type | Resolution Process |
|------------|---------------------|
| Missing vendor manual | Contact vendor/procurement; escalate if needed |
| Poor quality manual | Request replacement from vendor |
| Missing drawing | Coordinate with discipline lead |
| TM rejection | Review comments; revise and resubmit |
