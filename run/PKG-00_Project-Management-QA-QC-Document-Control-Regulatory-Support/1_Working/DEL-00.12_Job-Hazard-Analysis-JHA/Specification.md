## 1. Scope

### 1.1 Purpose

This specification defines the requirements for Job Hazard Analysis (JHA) documents to be developed by the Engineering Contractor for the Puget Sound Optimization Project (PSO). JHAs systematically identify workplace hazards associated with specific work activities and define controls to mitigate those hazards.

### 1.2 Applicability

This specification applies to:
- All work sites within the PSO scope
- All work activities performed at each site
- Pre-commencement safety documentation

### 1.3 Boundaries

**In Scope:**
- Hazard identification for work activities
- Risk assessment
- Control measure definition
- Worker communication and acknowledgment

**Out of Scope:**
- Completion status documentation (covered by CMR - DEL-00.11)
- Formal risk assessments/HAZOPs (covered by DEL-00.08, DEL-00.09)
- Detailed safety procedures (may be referenced in JHA)
