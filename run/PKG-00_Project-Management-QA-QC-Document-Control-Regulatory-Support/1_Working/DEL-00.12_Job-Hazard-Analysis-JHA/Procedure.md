## 1. Purpose

### 1.1 Objective

This procedure defines the step-by-step process for developing, reviewing, approving, and implementing Job Hazard Analyses (JHAs) for the Puget Sound Optimization Project (PSO).

### 1.2 Scope

This procedure covers:
- JHA template setup and activity identification
- Hazard identification and risk assessment
- Control measure development
- JHA review and approval
- Worker communication and implementation
- JHA revision and maintenance

## 3. Steps

### 3.1 Setup and Planning

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.1.1 | Obtain TM JHA template (or TM-approved format) | HSE Lead | Approved template/format |
| 3.1.2 | Identify work activities requiring JHA | Site Supervisor | Activity list |
| 3.1.3 | Prioritize JHAs based on risk level and schedule | HSE Lead | Prioritized JHA list |
| 3.1.4 | Establish JHA numbering convention | Document Controller | Numbering convention |
| 3.1.5 | Coordinate with CMR development (DEL-00.11) | HSE Lead | Aligned schedules |

### 3.2 JHA Team Assembly

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.2.1 | Identify workers who will perform the task | Site Supervisor | Worker list |
| 3.2.2 | Schedule JHA development session | HSE Lead | Session scheduled |
| 3.2.3 | Assemble JHA team (supervisor, workers, HSE) | Site Supervisor | Team assembled |
| 3.2.4 | Gather reference materials (procedures, drawings, specs) | HSE Lead | Reference materials |

### 3.3 Task Analysis

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.3.1 | Observe or walk through the task (if possible) | JHA Team | Task observation notes |
| 3.3.2 | Break task into sequential steps | JHA Team | Task step list |
| 3.3.3 | Review task steps for completeness | JHA Team | Validated step list |
| 3.3.4 | Document task steps in JHA form | HSE Lead | JHA task section complete |

### 3.4 Hazard Identification

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.4.1 | For each task step, ask "What could go wrong?" | JHA Team | Initial hazard list |
| 3.4.2 | Consider energy sources, environmental factors, human factors | JHA Team | Expanded hazard list |
| 3.4.3 | Review incident history for similar tasks | HSE Lead | Historical hazard data |
| 3.4.4 | Document hazards for each task step | HSE Lead | JHA hazard section complete |
| 3.4.5 | Assess risk level (severity x probability) for each hazard (see Datasheet Section 2.3, Specification JHA-C-007) | JHA Team | Risk ratings assigned |

### 3.5 Control Development

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.5.1 | For each hazard, identify control measures | JHA Team | Control measures list |
| 3.5.2 | Prioritize controls using hierarchy: elimination > substitution > engineering > administrative > PPE | HSE Lead | Prioritized controls |
| 3.5.3 | Assign responsible party for each control | Site Supervisor | Responsibilities assigned |
| 3.5.4 | Verify controls are feasible and available | Site Supervisor | Controls validated |
| 3.5.5 | Document controls in JHA form | HSE Lead | JHA control section complete |

### 3.6 JHA Review and Approval

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.6.1 | Review completed JHA for completeness | HSE Lead | Review complete |
| 3.6.2 | Verify consistency with CMR (DEL-00.11) | HSE Lead | Consistency verified |
| 3.6.3 | Obtain Site Supervisor approval | Site Supervisor | Supervisor sign-off |
| 3.6.4 | Register JHA in document control | Document Controller | JHA registered |

### 3.7 Worker Communication

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.7.1 | Conduct JHA briefing with all workers performing the task | Site Supervisor | Briefing conducted |
| 3.7.2 | Walk through each task step, hazard, and control | Site Supervisor | Worker understanding confirmed |
| 3.7.3 | Address worker questions and concerns | Site Supervisor | Questions resolved |
| 3.7.4 | Obtain worker acknowledgment signatures (see Specification JHA-C-010, JHA-F-004) | Site Supervisor | Acknowledgment records |
| 3.7.5 | Post or provide JHA copies at work location | Site Supervisor | JHA available in field |

### 3.8 Implementation and Monitoring

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.8.1 | Verify controls are implemented before task begins | Site Supervisor | Controls in place |
| 3.8.2 | Monitor work execution for compliance with JHA | Site Supervisor | Ongoing monitoring |
| 3.8.3 | Address deviations or new hazards immediately | Site Supervisor / HSE Lead | Issues addressed |
| 3.8.4 | Stop work if significant new hazard identified | Any worker | Work stopped; JHA revised |

### 3.9 JHA Revision

| Step | Action | Responsible | Output |
|------|--------|-------------|--------|
| 3.9.1 | Identify need for JHA revision (condition change, new hazard, incident) | Site Supervisor / HSE Lead | Revision trigger |
| 3.9.2 | Reconvene JHA team to update analysis | HSE Lead | Revised JHA |
| 3.9.3 | Follow Steps 3.6-3.7 for review, approval, and communication | Per steps | Approved revision |
| 3.9.4 | Archive previous JHA version | Document Controller | Archived version |
| 3.9.5 | Re-brief workers on revised JHA | Site Supervisor | Workers updated |

## 4. Verification

### 4.1 Quality Control Checklist

| Check Item | Verification Method | Pass Criteria |
|------------|---------------------|---------------|
| All task steps documented | Review JHA | Steps complete and sequential |
| Hazards identified for each step | Review JHA | At least one hazard per step |
| Controls defined for each hazard | Review JHA | At least one control per hazard |
| Controls are specific and feasible | Review with workers | Workers confirm feasibility |
| Responsible parties assigned | Review JHA | All controls have owners |
| Approval signatures present | Review JHA | Required signatures obtained |
| Worker acknowledgment records | Review records | All workers signed |
| JHA registered in document control | Check register | Entry exists |

### 4.2 Review Requirements

| Milestone | Reviewer | Approval |
|-----------|----------|----------|
| JHA Draft | HSE Lead | Technical review |
| JHA Finalization | Site Supervisor | Operational approval |
| Worker Briefing | Workers | Understanding confirmed |

### 4.3 Issue Resolution

| Issue Type | Resolution Process |
|------------|---------------------|
| Missing hazard identified | Stop work if immediate risk; update JHA and re-brief |
| Control not feasible | Identify alternative control; update JHA |
| Worker concerns | Address concerns before proceeding; update JHA if needed |
