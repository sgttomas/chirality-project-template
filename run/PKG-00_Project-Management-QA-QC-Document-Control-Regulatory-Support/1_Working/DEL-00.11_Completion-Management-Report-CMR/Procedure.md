## 1. Purpose

### 1.1 Objective

This procedure defines the step-by-step process for developing, reviewing, submitting, and obtaining approval for Completion Management Reports (CMRs) for the Puget Sound Optimization Project (PSO).

### 1.2 Scope

This procedure covers:
- CMR template setup and preparation
- Completion status data collection
- CMR drafting and internal review
- TM submission and approval process
- CMR updates and revisions
