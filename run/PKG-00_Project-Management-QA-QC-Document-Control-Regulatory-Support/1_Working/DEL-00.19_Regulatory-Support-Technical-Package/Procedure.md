## 1. Purpose

### 1.1 Objective

This procedure defines the step-by-step process for receiving, preparing, reviewing, and delivering regulatory support technical documentation for the Puget Sound Optimization Project (PSO).

### 1.2 Scope

This procedure covers:
- Regulatory support request management
- Technical document preparation and stamping
- Equipment registration coordination
- Safety Codes Officer engagement
- Regulatory query response
