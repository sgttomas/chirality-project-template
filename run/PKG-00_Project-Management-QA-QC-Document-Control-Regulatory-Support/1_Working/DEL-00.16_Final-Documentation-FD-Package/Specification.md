## 1. Scope

### 1.1 Purpose

This specification defines the requirements for the Final Documentation (FD) Package to be assembled and delivered by the Engineering Contractor for the Puget Sound Optimization Project (PSO). The FD Package shall constitute the complete, indexed collection of all engineering documentation required to support procurement, construction, commissioning, operations, and maintenance.

### 1.2 Applicability

This specification applies to:
- All FEED and Detailed Design deliverables across all disciplines (PKG-00 through PKG-09)
- All document types including drawings, calculations, reports, specifications, data sheets, lists, registers, and data books
- Both electronic native files and PDF exports
- Index and cross-reference documentation

### 1.3 Boundaries

**In Scope:**
- All Engineering Contractor-produced documents at IFC/As-Built status
- Document organization and indexing
- Cross-referencing and traceability
- Native file and PDF compilation
- Delivery per TM requirements

**Out of Scope:**
- TM-originated documents (reference only)
- Third-party documents not produced by Engineering Contractor
- Construction field records (unless specifically assigned)
