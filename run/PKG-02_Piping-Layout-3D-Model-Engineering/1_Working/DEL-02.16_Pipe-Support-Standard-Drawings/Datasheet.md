## 3. Scope Definition

### 3.1 Included Scope
- Standard drawings for each type of pipe support used in the PSO scope
- Support types for new booster pump station piping (5 x VS6-type pumps)
- Support types for new 5x6 manifold piping
- Support types for associated infrastructure piping at SW corner of TK-102
- Support details compatible with stress analysis requirements per DEL-02.14
- Support details compatible with 3D model integration per DEL-02.28

### 3.2 Excluded Scope
- Site-specific support location plans (covered by DEL-02.17)
- Existing support modifications unless specifically required for new tie-ins
- Laurel Pump Station scope (explicitly excluded per SOW)
