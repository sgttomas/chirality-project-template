# Procedure: Module & Prefabrication List

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for creating, maintaining, and issuing the Module & Prefabrication List for the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to all modules, skids, and prefabricated components within the PSO project piping scope at Sumas Tank Farm.

## 2. Prerequisites

### 2.1 Input Documents

| Document | Source | Status Required |
|----------|--------|-----------------|
| 3D Model | DEL-02.28 | 30% minimum |
| Plot Plan | DEL-02.08 | IFD minimum |
| Equipment List | DEL-03.01 | Approved |
| GA Drawings | DEL-02.12 | Draft minimum |
| Piping Isometrics | DEL-02.13 | Draft minimum |
| Construction Execution Plan | Construction | Approved strategy |

### 2.2 Resources Required
- Piping Engineer (Lead)
- Construction Planner
- Logistics Coordinator
- 3D Model Coordinator

### 2.3 Training Requirements
- Understanding of modularization principles
- Knowledge of transport regulations
- Familiarity with project 3D model

## 3. Steps

### 3.1 Strategy Development

| Step | Action | Responsible |
|------|--------|-------------|
| 3.1.1 | Review construction execution strategy | Piping Lead |
| 3.1.2 | Identify site access and crane constraints | Construction Planner |
| 3.1.3 | Determine transport limits (standard vs permit) | Logistics Coordinator |
| 3.1.4 | Establish modularization criteria | Piping Lead |
| 3.1.5 | Document strategy in execution plan | Piping Lead |

### 3.2 Module Identification

| Step | Action | Responsible |
|------|--------|-------------|
| 3.2.1 | Review 3D model for modularization opportunities | Piping Engineer |
| 3.2.2 | Identify equipment skids suitable for pre-assembly | Piping Engineer |
| 3.2.3 | Identify pipe rack sections for prefabrication | Piping Engineer |
| 3.2.4 | Identify pipe spools for shop fabrication | Piping Engineer |
| 3.2.5 | Assign unique module/prefab IDs | Piping Designer |

### 3.3 Data Population

| Step | Action | Responsible |
|------|--------|-------------|
| 3.3.1 | Extract dimensions from 3D model | 3D Coordinator |
| 3.3.2 | Calculate/estimate weights | Piping Engineer |
| 3.3.3 | Assign fabrication locations | Piping Lead |
| 3.3.4 | Determine delivery priorities | Construction Planner |
| 3.3.5 | Establish ROS dates based on schedule | Construction Planner |
| 3.3.6 | Link associated drawings | Piping Designer |

### 3.4 Review and Validation

| Step | Action | Responsible |
|------|--------|-------------|
| 3.4.1 | Review module sizes against transport limits | Logistics Coordinator |
| 3.4.2 | Review weights against lifting capacity | Construction Planner |
| 3.4.3 | Review delivery sequence against schedule | Construction Planner |
| 3.4.4 | Constructability review with Construction | Construction Lead |
| 3.4.5 | Technical review by Piping Lead | Piping Lead |

### 3.5 Issue and Maintenance

| Step | Action | Responsible |
|------|--------|-------------|
| 3.5.1 | Submit for internal review | Piping Engineer |
| 3.5.2 | Incorporate review comments | Piping Designer |
| 3.5.3 | Submit for Owner review | Document Control |
| 3.5.4 | Issue approved revision | Document Control |
| 3.5.5 | Update list as design progresses | Piping Engineer |
| 3.5.6 | Track fabrication and delivery status | Logistics Coordinator |

## 4. Verification

### 4.1 Quality Checks

| Check | Criteria | Method |
|-------|----------|--------|
| Completeness | All prefab items identified | Cross-reference to isometrics |
| Dimensions | Within transport limits | Limit comparison |
| Weights | Reasonable estimates with basis | Engineering review |
| Schedule | ROS dates aligned with construction | Schedule review |

### 4.2 Review and Approval

| Role | Responsibility |
|------|----------------|
| Piping Designer | Prepare and self-check |
| Piping Engineer | Technical review |
| Construction Planner | Constructability review |
| Piping Lead | Approve for issue |
| TM Representative | Owner review and approval |

### 5.1 Records Generated

| Aspect | Requirement |

### 5.3 Ongoing Maintenance
- Update list as isometrics are issued/revised
- Track fabrication progress weekly
- Track shipment and delivery status

- Archive completed module records
