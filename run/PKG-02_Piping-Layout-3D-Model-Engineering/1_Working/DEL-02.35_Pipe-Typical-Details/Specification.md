# Specification: Pipe Typical Details

## 1. Scope

### 1.1 Purpose
This specification defines the requirements for developing Pipe Typical Detail drawings for the Puget Sound Optimization Project, providing standardized installation details for piping components, supports, and accessories.

### 1.2 Applicability
- All new piping systems within PSO project scope
- Pipe support installations
- Branch connections and specialty items
- Valve and instrument installations
- Insulation and heat trace details
- Underground piping details

### 1.3 Exclusions
- Equipment-specific vendor details (provided by equipment vendor)
- Instrument-specific installation details (covered under Instrumentation package)
- Structural steel details (covered under Civil/Structural package)
- Electrical conduit supports

## 2. Requirements

### 2.1 Content Requirements
The Pipe Typical Details package shall include:

| Requirement | Description |
|-------------|-------------|
| REQ-01 | Standard pipe support details covering all support types used on project |
| REQ-02 | Branch connection details for all connection types |
| REQ-03 | Valve installation details showing orientation, access, and clearances |
| REQ-04 | Specialty item installation details (meters, strainers, filters) |
| REQ-05 | Drain and vent arrangement details |
| REQ-06 | Insulation termination and protection details |
| REQ-07 | Heat trace routing and connection details |
| REQ-08 | Underground piping details (burial, coating, CP) |
| REQ-09 | Penetration details for walls and floors |
| REQ-10 | Project-specific special details as required |

### 2.2 Format Requirements
- CAD drawings in native format (DWG) plus PDF
- Standard sheet sizes per project document standard
- Clear dimensioning with tolerances where applicable
- Material callouts per PMC Index (DEL-02.25)
- Notes for fabrication and installation requirements
- Reference to applicable codes and standards

### 2.3 Technical Requirements
- Details shall comply with applicable codes (ASME B31.3, CSA Z662)
- Support details shall comply with MSS SP-58 and MSS SP-69
- Details shall be compatible with TM existing standards where applicable
- Details shall address both shop and field fabrication/installation

### 2.4 Coordination Requirements
- Typical details shall be referenced on piping isometrics (DEL-02.13)
- Support details shall coordinate with support location plans (DEL-02.17)
- Specialty item details shall align with datasheets (DEL-02.20)
- Details shall be incorporated into CWPs and FIWPs
