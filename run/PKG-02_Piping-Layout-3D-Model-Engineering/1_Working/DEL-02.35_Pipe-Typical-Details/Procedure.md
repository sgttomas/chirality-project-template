# Procedure: Pipe Typical Details

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for developing, reviewing, and issuing Pipe Typical Detail drawings for the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to all pipe typical details within the PSO project scope at Sumas Tank Farm.
