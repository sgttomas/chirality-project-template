# Procedure: Piping Key Plan

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for creating, maintaining, and issuing the Piping Key Plan for the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to the overall site key plan for the PSO project at Sumas Tank Farm.

## 2. Prerequisites

### 2.1 Input Documents
| Document | Source | Status Required |
|----------|--------|-----------------|
| Site Survey | Civil/Survey | Completed |
| Plot Plan | DEL-02.08 | Draft minimum |
| Existing Site Drawings | TM-provided | Latest revision |
| 3D Model | DEL-02.28 | Preliminary |
| CAD Standards | TM Standard | Confirmed (**TBD**) |

### 2.2 Resources Required
- Lead Piping Designer
- CAD Technician
- Access to 3D model
- CAD template per TM standards

### 2.3 Training Requirements
- Proficiency in CAD software (AutoCAD or equivalent)
- Familiarity with TM CAD/drawing standards
- Document control procedures

## 4. Verification

### 4.1 Quality Checks

| Check | Criteria | Method |
|-------|----------|--------|
| Completeness | All PSO areas covered | Visual review |
| Consistency | Grid matches survey/Civil | Coordinate check |
| Accuracy | Equipment locations match Plot Plan | Overlay comparison |
| Index | All drawings listed with correct coverage | Cross-reference |

### 4.2 Review and Approval

| Role | Responsibility |
|------|----------------|
| CAD Technician | Prepare and self-check |
| Piping Designer | Technical coordination |
| Piping Lead | Approve for issue |
| TM Representative | Owner review and approval |
