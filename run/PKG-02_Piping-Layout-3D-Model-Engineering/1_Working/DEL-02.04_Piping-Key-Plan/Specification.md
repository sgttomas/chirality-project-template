# Specification: Piping Key Plan

## 1. Scope

### 1.1 Purpose
This specification defines the requirements for developing and maintaining the Piping Key Plan for the Puget Sound Optimization Project at Sumas Tank Farm.

### 1.2 Applicability
- Overall site piping layout reference
- Drawing sheet index for detailed piping plans
- Area/grid reference system for all piping deliverables

### 1.3 Exclusions
- Detailed piping routing (covered in Site Plan and GA drawings)
- Equipment details (covered in Equipment Layout Plans)
- Instrumentation (covered under Instrumentation package)

## 2. Requirements

### 2.1 Content Requirements
The Piping Key Plan shall include:

| Requirement | Description |
|-------------|-------------|
| REQ-01 | Site boundary and property lines |
| REQ-02 | Coordinate grid system |
| REQ-03 | Match-line references for adjacent drawings |
| REQ-04 | Major equipment locations (tanks, pumps, manifolds) |
| REQ-05 | Primary piping corridors/routes |
| REQ-06 | Area/zone designations |
| REQ-07 | Drawing sheet index with coverage areas |
| REQ-08 | North arrow and scale bar |
| REQ-09 | Legend for all symbols |
| REQ-10 | Standard title block |

### 2.2 Format Requirements
- CAD drawing format (DWG native, PDF for distribution)
- D-size or E-size sheet (**ASSUMPTION** - confirm with TM)
- Scale appropriate for overall site coverage
- Comply with TM CAD standards (**TBD**)

### 2.3 Accuracy Requirements
- Grid/coordinate system consistent with survey data
- Match-lines aligned with detailed drawing coverage
- Equipment locations consistent with plot plan

## 4. Verification

### 4.1 Verification Methods
| Method | Description |
|--------|-------------|
| Review | Technical review by Lead Piping Engineer |
| Cross-check | Verification against Plot Plan and Area Plans |
| Coordination | Verify match-line alignment with detailed drawings |
| Audit | Document control audit for completeness |

### 4.2 Acceptance Criteria
- All PSO project areas represented
- Grid system consistent with site survey
- Match-lines correspond to actual detailed drawing coverage
- Drawing sheet index complete and accurate
- Owner review and approval
