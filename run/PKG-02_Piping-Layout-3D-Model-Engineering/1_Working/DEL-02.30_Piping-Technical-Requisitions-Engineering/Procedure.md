# Procedure: DEL-02.30 Piping Technical Requisitions (Engineering)

## Purpose

### Objective
This procedure defines the steps for developing, reviewing, approving, and issuing piping technical requisitions for the Puget Sound Optimization Project at Sumas Tank Farm, ensuring complete and accurate technical packages for procurement.

### Scope
Applies to all technical requisitions for piping materials, valves, specialty items, pipe supports, insulation, and related equipment within PKG-02 scope.
