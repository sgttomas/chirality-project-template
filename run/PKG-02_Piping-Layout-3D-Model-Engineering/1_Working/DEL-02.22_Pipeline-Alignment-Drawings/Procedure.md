# Procedure: Pipeline Alignment Drawings

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for creating, reviewing, and issuing pipeline alignment drawings for the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to all new pipeline segments within the PSO project scope at Sumas Tank Farm.

## 2. Prerequisites

### 2.1 Input Documents

| Document | Source | Status Required |
|----------|--------|-----------------|
| Survey Data | Survey Contractor | Verified and approved |
| Route Selection Study | Engineering | Approved route |
| Geotechnical Report | Geotech Contractor | Available |
| Environmental Permits | Regulatory | Approved constraints |
| P&IDs | DEL-01.03 | IFD minimum |
| Utility Crossing Info | Third Parties | Confirmed locations |

### 2.2 Resources Required
- Pipeline Engineer (Lead)
- Civil Engineer (grading/drainage)
- CAD Operator
- Survey Coordinator

### 2.3 Training Requirements
- Pipeline design per CSA Z662 / ASME B31.4
- CAD drawing standards
- Survey data interpretation

## 3. Steps

### 3.1 Data Collection

| Step | Action | Responsible |
|------|--------|-------------|
| 3.1.1 | Obtain survey control data and topographic survey | Survey Coordinator |
| 3.1.2 | Obtain geotechnical information along route | Pipeline Engineer |
| 3.1.3 | Collect utility crossing information | Pipeline Engineer |
| 3.1.4 | Gather environmental constraint mapping | Pipeline Engineer |
| 3.1.5 | Confirm approved route alignment | Pipeline Lead |

### 3.2 Horizontal Alignment

| Step | Action | Responsible |
|------|--------|-------------|
| 3.2.1 | Establish pipeline centerline in CAD | CAD Operator |
| 3.2.2 | Set up stationing along centerline | CAD Operator |
| 3.2.3 | Define horizontal curves (PI, radius, tangent) | Pipeline Engineer |
| 3.2.4 | Verify alignment stays within ROW | Pipeline Engineer |
| 3.2.5 | Identify all crossing locations | Pipeline Engineer |

### 3.3 Vertical Alignment

| Step | Action | Responsible |
|------|--------|-------------|
| 3.3.1 | Import ground profile from survey | CAD Operator |
| 3.3.2 | Set pipeline depth based on cover requirements | Pipeline Engineer |
| 3.3.3 | Define vertical curves (overbend/sagbend) | Pipeline Engineer |
| 3.3.4 | Verify crossing clearances | Pipeline Engineer |
| 3.3.5 | Coordinate with Civil for grading changes | Civil Engineer |

### 3.4 Drawing Production

| Step | Action | Responsible |
|------|--------|-------------|
| 3.4.1 | Prepare key plan/index sheet | CAD Operator |
| 3.4.2 | Prepare plan and profile sheets | CAD Operator |
| 3.4.3 | Prepare typical cross-section sheet | CAD Operator |
| 3.4.4 | Prepare crossing detail sheets | CAD Operator |
| 3.4.5 | Prepare tie-in detail sheets | CAD Operator |
| 3.4.6 | Add notes, legends, and title blocks | CAD Operator |

### 3.5 Crossing Details

| Step | Action | Responsible |
|------|--------|-------------|
| 3.5.1 | Detail each road crossing with method | Pipeline Engineer |
| 3.5.2 | Detail each utility crossing with clearances | Pipeline Engineer |
| 3.5.3 | Detail any watercourse crossings | Pipeline Engineer |
| 3.5.4 | Obtain crossing agreements/permits | Pipeline Lead |

### 3.6 Review and Issue

| Step | Action | Responsible |
|------|--------|-------------|
| 3.6.1 | Self-check drawings for completeness | CAD Operator |
| 3.6.2 | Technical review by Pipeline Engineer | Pipeline Engineer |
| 3.6.3 | Interdisciplinary review (Civil, Environmental) | IDC Coordinator |
| 3.6.4 | Incorporate review comments | CAD Operator |
| 3.6.5 | Pipeline Lead approval | Pipeline Lead |
| 3.6.6 | Submit for Owner review | Document Control |
| 3.6.7 | Issue approved drawings | Document Control |

## 4. Verification

### 4.1 Quality Checks

| Check | Criteria | Method |
|-------|----------|--------|
| Survey Accuracy | Coordinates match survey data | Spot check |
| Cover Depth | Meets minimum requirements | Profile review |
| Crossing Clearances | Meet code/agreement requirements | Detail review |
| Bend Radius | Within pipe limits | Calculation check |

### 4.2 Review and Approval

| Role | Responsibility |
|------|----------------|
| CAD Operator | Prepare and self-check |
| Pipeline Engineer | Technical review |
| Civil Engineer | Grading/drainage review |
| Pipeline Lead | Approve for issue |
| TM Representative | Owner review and approval |

## 5. Records

### 5.1 Records Generated

| Record | Format | Retention |
|--------|--------|-----------|
| Alignment Drawings | PDF + DWG | Project life + permanent (as-built) |
| Survey Data | Digital survey files | Project life + 10 years |
| Crossing Agreements | PDF | Project life + permanent |
| Review Records | PDF/Email | Project life |

### 5.2 Document Control

| Aspect | Requirement |
|--------|-------------|
| **Numbering** | Per TM document numbering (**TBD**) |
| **Revision Control** | Track all revisions with cloud markup |
| **Distribution** | Per project distribution matrix |
| **Storage** | Project document management system |

### 5.3 As-Built Requirements
- Update alignment drawings with as-built survey data
- Document any field changes to alignment
- Record actual crossing locations and depths
- Maintain permanent record for pipeline integrity management
