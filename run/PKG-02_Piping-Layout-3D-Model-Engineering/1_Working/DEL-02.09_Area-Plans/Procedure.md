## 1. Purpose
This procedure defines the step-by-step process for developing the Area Plan drawing(s) for the Puget Sound Optimization Project (PSO) at Sumas Tank Farm.

## 2. Scope
This procedure applies to the development, review, and issuance of area plan drawings from FEED through Detailed Design phases.

## 3. Responsibilities

| Role | Responsibilities |
|------|-----------------|
| Piping Lead | Overall deliverable ownership, technical review, approval |
| Piping Designer/Engineer | Drawing development, coordination, revisions |
| CAD Technician | Drawing production per standards |
| 3D Modeler | 3D model development and plan extraction |
| Civil/Structural Lead | Foundation and structural coordination |
| Electrical Lead | Area classification and electrical coordination |
| I&C Lead | Instrumentation location coordination |
| Document Controller | Document numbering, revision control, issuance |

## 4. Prerequisites

Before commencing drawing development:

| Prerequisite | Source | Status |
|--------------|--------|--------|

## 5. Procedure

### Step 1: Gather Input Data
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 1.1 | Obtain overall plot plan (DEL-02.08) for context |
| 1.2 | Obtain equipment list (DEL-03.01) for complete equipment coverage |
| 1.3 | Obtain preliminary equipment datasheets (DEL-03.02) for dimensions |
| 1.4 | Obtain 3D model extract for current layout |
| 1.5 | Obtain foundation layout (DEL-04.17) for structural coordination |
| 1.6 | Obtain TM CAD standards and drawing templates |
| 1.7 | Document any gaps in input data |

**Output:** Collected input data package, gap list (if applicable)

---

### Step 2: Define Area Coverage
**Responsible:** Piping Designer/Engineer, Piping Lead

| Action | Details |
|--------|---------|
| 2.1 | Review project layout and identify distinct areas |
| 2.2 | Define area boundaries considering functional groupings |
| 2.3 | Determine number of sheets required |
| 2.4 | Select appropriate scale for area detail (typically 1:100) |
| 2.5 | Plan match line locations between sheets |
| 2.6 | Establish sheet numbering sequence |
| 2.7 | Document area definitions for team coordination |

**Output:** Area coverage plan, sheet list

---

### Step 3: Set Up Drawing Files
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 3.1 | Set up drawing files per TM CAD standards |
| 3.2 | Import coordinate grid from site survey |
| 3.3 | Establish drawing scale and viewport arrangement |
| 3.4 | Set up layer structure per area plan requirements |
| 3.5 | Prepare title block with preliminary document numbers |
| 3.6 | Set up key plan template for each sheet |

**Output:** Drawing templates ready for content

---

### Step 4: Develop Base Content
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 4.1 | Import/extract plan views from 3D model |
| 4.2 | Add or verify equipment footprints with tags |
| 4.3 | Add foundation outlines from structural drawings |
| 4.4 | Add containment walls and boundaries |
| 4.5 | Add building outlines and entrances |
| 4.6 | Add coordinate grid labels |
| 4.7 | Establish match lines between sheets |

**Output:** Base drawings with equipment and structures

---

### Step 5: Add Piping and Equipment Details
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 5.1 | Add major piping routing from 3D model |
| 5.2 | Add line numbers and flow direction arrows |
| 5.3 | Add valve locations with tags |
| 5.4 | Add instrument connection points |
| 5.5 | Add pipe support location references |
| 5.6 | Verify equipment nozzle orientations |
| 5.7 | Add VFD/motor and auxiliary equipment locations |

**Output:** Drawings with piping and equipment details

---

### Step 6: Add Access and Safety Features
**Responsible:** Piping Designer, with Safety input

| Action | Details |
|--------|---------|
| 6.1 | Add access walkways and platforms |
| 6.2 | Verify minimum clearances for maintenance |
| 6.3 | Add emergency egress routes |
| 6.4 | Add safety equipment locations (if applicable) |
| 6.5 | Add hazardous area boundaries (or reference) |
| 6.6 | Verify valve access positions |

**Output:** Drawings with access and safety features

---

### Step 7: Complete Drawing Annotations
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 7.1 | Add dimensions for key equipment coordinates |
| 7.2 | Add drawing references and cross-references |
| 7.3 | Complete legend with all symbols used |
| 7.4 | Add general notes |
| 7.5 | Complete key plans showing sheet coverage |
| 7.6 | Add north arrow and scale notation |
| 7.7 | Complete title block information |

**Output:** Annotated draft drawings

---

### Step 8: Discipline Coordination
**Responsible:** Piping Designer, Design Coordinator

| Action | Details |
|--------|---------|
| 8.1 | Issue drafts for Civil/Structural review |
| 8.2 | Issue drafts for Electrical review |
| 8.3 | Issue drafts for I&C review |
| 8.4 | Issue drafts for Mechanical review |
| 8.5 | Collect and consolidate coordination comments |
| 8.6 | Resolve conflicts through coordination meetings |
| 8.7 | Incorporate agreed changes |
| 8.8 | Update 3D model as required |

**Output:** Coordinated drawings, meeting notes

---

### Step 9: Internal Quality Check
**Responsible:** Piping Lead, Piping Designer

| Action | Details |
|--------|---------|
| 9.1 | Self-check against Specification requirements |
| 9.2 | Verify complete equipment coverage |
| 9.3 | Verify consistency with 3D model |
| 9.4 | Verify match line coordination |
| 9.5 | Check CAD standards compliance |
| 9.6 | Discipline check by Piping Lead |
| 9.7 | Document and resolve check comments |

**Output:** Checked drawings, resolved comment log

---

### Step 10: Issue for Review (IFR)
**Responsible:** Piping Lead, Document Controller

| Action | Details |
|--------|---------|
| 10.1 | Finalize drawings for IFR issue |
| 10.2 | Assign official document numbers per DEL-00.02 |
| 10.3 | Complete transmittal documentation |
| 10.4 | Submit to TM for review per SOW requirements |
| 10.5 | Log issue in MDR (DEL-00.01) |

**Output:** IFR drawings issued, transmittal record

---

### Step 11: Comment Resolution
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 11.1 | Receive and log TM review comments |
| 11.2 | Review and categorize comments |
| 11.3 | Prepare responses/resolutions for each comment |
| 11.4 | Coordinate clarifications with TM as needed |
| 11.5 | Implement accepted changes in drawings |
| 11.6 | Document comment resolution in log |

**Output:** Comment response log, updated drawings

---

### Step 12: Issue for Design (IFD)
**Responsible:** Piping Lead, Document Controller

| Action | Details |
|--------|---------|
| 12.1 | Incorporate all resolved comments |
| 12.2 | Final quality check |
| 12.3 | Update revision block and issue status |
| 12.4 | Issue IFD drawings |
| 12.5 | Update MDR |

**Output:** IFD drawings issued

---

### Step 13: Detailed Design Updates
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 13.1 | Incorporate vendor data updates |
| 13.2 | Incorporate HAZOP action item resolutions |
| 13.3 | Update for design changes per change control |
| 13.4 | Maintain consistency with 3D model and related drawings |
| 13.5 | Progress to IFC |

**Output:** IFC drawings

---

### Step 14: Native File Submission
**Responsible:** CAD Technician, Document Controller

| Action | Details |
|--------|---------|
| 14.1 | Prepare native CAD files per SOW-0259 requirements |
| 14.2 | Verify file naming convention compliance |
| 14.3 | Submit native files with PDF packages |
| 14.4 | Confirm receipt by TM document control |

**Output:** Native files submitted
