# Procedure: Hydrostatic Test Packages

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for creating, reviewing, and issuing hydrostatic test packages for the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to all new piping within the PSO project scope at Sumas Tank Farm requiring hydrostatic testing.

## 2. Prerequisites

### 2.1 Input Documents

| Document | Source | Status Required |
|----------|--------|-----------------|
| Piping Isometrics | DEL-02.13 | IFD minimum |
| Line List | DEL-02.01 | Approved |
| Line Designation Table | DEL-02.18 | IFD minimum |
| P&IDs | DEL-01.03 | IFC |
| Design Basis | DEL-01.01 | Approved |
| Pipe Support Drawings | DEL-02.16 | Draft minimum |

### 2.2 Resources Required
- Piping Engineer (Lead)
- Piping Designer
- CAD Operator
- Construction input (for constructability)

### 2.3 Training Requirements
- Knowledge of hydrotest code requirements (ASME B31.3, CSA Z662)
- Familiarity with project piping specifications
- CAD drawing standards

## 3. Steps

### 3.1 Planning

| Step | Action | Responsible |
|------|--------|-------------|
| 3.1.1 | Obtain current P&IDs and piping isometrics | Piping Engineer |
| 3.1.2 | Review Line Designation Table for test requirements | Piping Engineer |
| 3.1.3 | Identify all lines requiring hydrotesting | Piping Engineer |
| 3.1.4 | Develop preliminary test boundary plan | Piping Engineer |
| 3.1.5 | Review boundary plan with Construction | Piping Lead |

### 3.2 Boundary Definition

| Step | Action | Responsible |
|------|--------|-------------|
| 3.2.1 | Define logical test boundaries per system/area | Piping Engineer |
| 3.2.2 | Identify isolation points and blind requirements | Piping Engineer |
| 3.2.3 | Identify equipment requiring protection/isolation | Piping Engineer |
| 3.2.4 | Assign unique test package numbers | Piping Designer |
| 3.2.5 | Create test package index | Piping Designer |

### 3.3 Test Pressure Calculation

| Step | Action | Responsible |
|------|--------|-------------|
| 3.3.1 | Identify design pressure for each line in boundary | Piping Engineer |
| 3.3.2 | Identify weakest component rating | Piping Engineer |
| 3.3.3 | Calculate test pressure per applicable code | Piping Engineer |
| 3.3.4 | Document calculation on standard form | Piping Engineer |
| 3.3.5 | Verify test pressure does not exceed any component rating | Piping Engineer |

### 3.4 Drawing Preparation

| Step | Action | Responsible |
|------|--------|-------------|
| 3.4.1 | Create test boundary drawing from isometric base | CAD Operator |
| 3.4.2 | Apply color coding per project standard | CAD Operator |
| 3.4.3 | Mark vent and drain locations | Piping Designer |
| 3.4.4 | Mark blind/isolation locations with sizes | Piping Designer |
| 3.4.5 | Add test pressure and package number annotation | Piping Designer |

### 3.5 Supporting Documentation

| Step | Action | Responsible |
|------|--------|-------------|
| 3.5.1 | Prepare vent/drain schedule | Piping Designer |
| 3.5.2 | Prepare blind/isolation list with sizes and ratings | Piping Designer |
| 3.5.3 | Prepare water supply and disposal plan | Piping Engineer |
| 3.5.4 | Identify temporary support requirements | Stress Engineer |
| 3.5.5 | Compile complete test package | Piping Designer |

### 3.6 Review and Issue

| Step | Action | Responsible |
|------|--------|-------------|
| 3.6.1 | Self-check package for completeness | Piping Designer |
| 3.6.2 | Technical review by Piping Engineer | Piping Engineer |
| 3.6.3 | Constructability review with Construction | Construction Lead |
| 3.6.4 | Incorporate review comments | Piping Designer |
| 3.6.5 | Piping Lead approval | Piping Lead |
| 3.6.6 | Submit for Owner review | Document Control |
| 3.6.7 | Issue approved package | Document Control |

## 4. Verification

### 4.1 Quality Checks

| Check | Criteria | Method |
|-------|----------|--------|
| Coverage | All new lines included in a test package | Cross-reference to Line List |
| Pressure | Test pressure correctly calculated | Calculation review |
| Boundaries | No gaps or overlaps | Package index review |
| Completeness | All package elements present | Checklist verification |

### 4.2 Review and Approval

| Role | Responsibility |
|------|----------------|
| Piping Designer | Prepare and self-check |
| Piping Engineer | Technical review |
| Construction Lead | Constructability review |
| Piping Lead | Approve for issue |
| TM Representative | Owner review and approval |
