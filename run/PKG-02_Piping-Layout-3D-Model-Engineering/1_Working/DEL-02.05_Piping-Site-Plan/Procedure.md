# Procedure: Piping Site Plan

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for creating, maintaining, and issuing the Piping Site Plan for the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to the piping site plan(s) for the PSO project at Sumas Tank Farm.

## 2. Prerequisites

### 2.1 Input Documents
| Document | Source | Status Required |
|----------|--------|-----------------|
| Key Plan | DEL-02.04 | Draft minimum |
| Plot Plan | DEL-02.08 | Draft minimum |
| P&IDs | DEL-01.03 | IFR minimum |
| Line List | DEL-02.01 | Draft minimum |
| 3D Model | DEL-02.28 | Preliminary routing |
| CAD Standards | TM Standard | Confirmed (**TBD**) |

### 2.2 Resources Required
- Piping Designer (Lead)
- CAD Technician
- 3D Model Coordinator
- Access to 3D model extraction tools

### 2.3 Training Requirements
- Proficiency in CAD software
- 3D model navigation and extraction
- TM CAD/drawing standards
- Document control procedures

## 3. Steps

### 3.1 Preparation

| Step | Action | Responsible |
|------|--------|-------------|
| 3.1.1 | Obtain latest Key Plan and Plot Plan | Piping Designer |
| 3.1.2 | Obtain TM CAD template and standards | CAD Technician |
| 3.1.3 | Confirm 3D model status with coordinator | Piping Designer |
| 3.1.4 | Verify P&IDs are at IFR status minimum | Piping Designer |
| 3.1.5 | Obtain Line List for reference | Piping Designer |

### 3.2 Drawing Setup

| Step | Action | Responsible |
|------|--------|-------------|
| 3.2.1 | Create new drawing from TM template | CAD Technician |
| 3.2.2 | Import/reference Plot Plan as underlay | CAD Technician |
| 3.2.3 | Set coordinate system per Key Plan | CAD Technician |
| 3.2.4 | Set up layer structure per standards | CAD Technician |
| 3.2.5 | Establish match-line references from Key Plan | CAD Technician |

### 3.3 Piping Route Development

| Step | Action | Responsible |
|------|--------|-------------|
| 3.3.1 | Extract plan view from 3D model | 3D Coordinator |
| 3.3.2 | Import extracted piping into drawing | CAD Technician |
| 3.3.3 | Add above-ground piping routes | Piping Designer |
| 3.3.4 | Add below-ground piping routes (dashed) | Piping Designer |
| 3.3.5 | Add pipe rack/sleeper locations | Piping Designer |
| 3.3.6 | Add line numbers to all piping | CAD Technician |

### 3.4 Content Completion

| Step | Action | Responsible |
|------|--------|-------------|
| 3.4.1 | Add tie-in point locations and references | Piping Designer |
| 3.4.2 | Add major valve locations | Piping Designer |
| 3.4.3 | Add equipment outlines and tags | CAD Technician |
| 3.4.4 | Add road crossing details | Piping Designer |
| 3.4.5 | Add north arrow, scale, and grid references | CAD Technician |
| 3.4.6 | Complete legend and notes | CAD Technician |

### 3.5 Verification

| Step | Action | Responsible |

| 3.5.2 | Cross-check line numbers against Line List | Piping Designer |
| 3.5.3 | Verify tie-in points against Tie-In List | Piping Designer |
| 3.5.4 | Coordinate underground with Civil | Piping Designer |
| 3.5.5 | Review by Lead Piping Engineer | Piping Lead |

## 4. Verification

### 4.1 Quality Checks

| Check | Criteria | Method |
|-------|----------|--------|
| Completeness | All piping in scope represented | P&ID cross-check |
| Model Consistency | Routes match 3D model | Overlay comparison |
| Line Numbers | Match Line List | Cross-reference |
| Tie-Ins | Match Tie-In List | Cross-reference |
| Coordination | No conflicts with utilities | Clash detection |

### 4.2 Review and Approval

| Role | Responsibility |
|------|----------------|
| CAD Technician | Prepare and self-check |
| Piping Designer | Technical review |
| 3D Coordinator | Model consistency check |
| Piping Lead | Approve for issue |
| TM Representative | Owner review and approval |
