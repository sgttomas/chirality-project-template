# Procedure: Tie-In List / Tie Point List

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for creating, maintaining, and issuing the Tie-In List for the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to all piping tie-in points within the PSO project scope at Sumas Tank Farm.

## 2. Prerequisites

### 2.1 Input Documents
| Document | Source | Status Required |
|----------|--------|-----------------|
| P&IDs | DEL-01.03 | IFD minimum |
| Line List | DEL-02.01 | Draft minimum |
| Existing As-Built Drawings | TM-provided | Latest revision |
| Site Survey Data | Civil/Survey | As available |
| Tie-In Numbering Standard | TM Standard | Confirmed (**TBD**) |

### 2.2 Resources Required
- Piping Engineer (Lead)
- Field Engineer (for verification)
- TM Operations Representative (for isolation review)
- Access to existing facility drawings

### 2.3 Training Requirements
- Familiarity with brownfield tie-in procedures
- Knowledge of live system isolation requirements
- Document control procedures

## 3. Steps

### 3.1 Preparation

| Step | Action | Responsible |
|------|--------|-------------|
| 3.1.1 | Obtain latest revision of all P&IDs | Piping Engineer |
| 3.1.2 | Obtain existing facility as-built drawings | Piping Engineer |
| 3.1.3 | Obtain approved Tie-In List template | Document Control |
| 3.1.4 | Confirm tie-in numbering convention with TM | Piping Lead |
| 3.1.5 | Schedule coordination meeting with TM Operations | Piping Lead |

### 3.2 Initial Population

| Step | Action | Responsible |
|------|--------|-------------|
| 3.2.1 | Identify all tie-in points from P&IDs | Piping Engineer |
| 3.2.2 | Enter tie-in numbers into template | Piping Designer |
| 3.2.3 | Enter new and existing line numbers | Piping Designer |
| 3.2.4 | Add tie-in descriptions | Piping Engineer |
| 3.2.5 | Enter P&ID references | Piping Designer |

### 3.3 Technical Data Population

| Step | Action | Responsible |
|------|--------|-------------|
| 3.3.1 | Enter connection size and rating | Piping Engineer |
| 3.3.2 | Enter connection type | Piping Engineer |
| 3.3.3 | Enter preliminary location from drawings | Piping Designer |
| 3.3.4 | Identify preliminary isolation requirements | Piping Engineer |
| 3.3.5 | Recommend execution method | Piping Engineer |

### 3.4 Field Verification

| Step | Action | Responsible |
|------|--------|-------------|
| 3.4.1 | Schedule field walkdown with TM | Field Engineer |
| 3.4.2 | Verify tie-in locations on site | Field Engineer |
| 3.4.3 | Confirm existing line/equipment conditions | Field Engineer |
| 3.4.4 | Obtain survey coordinates | Survey Team |
| 3.4.5 | Update list with field-verified data | Piping Engineer |

### 3.5 Operations Coordination

| Step | Action | Responsible |
|------|--------|-------------|
| 3.5.1 | Submit draft tie-in list to TM Operations | Piping Engineer |
| 3.5.2 | Review isolation requirements with Operations | Piping Lead |
| 3.5.3 | Confirm shutdown requirements and durations | Piping Lead |
| 3.5.4 | Incorporate Operations feedback | Piping Engineer |
| 3.5.5 | Obtain Operations sign-off on isolation plan | Piping Lead |

### 3.6 Issue and Maintenance

| Step | Action | Responsible |
|------|--------|-------------|
| 3.6.1 | Submit for internal review | Piping Engineer |
| 3.6.2 | Incorporate review comments | Piping Designer |

| 3.6.4 | Issue approved revision | Document Control |

## 4. Verification

### 4.1 Quality Checks

| Check | Criteria | Method |
|-------|----------|--------|
| Completeness | All P&ID tie-ins captured | Count verification |
| Uniqueness | No duplicate tie-in numbers | Excel duplicate check |
| Field Accuracy | Locations match field conditions | Walkdown verification |
| Operations Review | Isolation requirements approved | Sign-off record |

### 4.2 Review and Approval

| Role | Responsibility |
|------|----------------|
| Piping Designer | Prepare and self-check |
| Piping Engineer | Technical review |
| Field Engineer | Field verification |
| TM Operations | Isolation/shutdown approval |
| Piping Lead | Approve for issue |
| TM Representative | Owner review and approval |
