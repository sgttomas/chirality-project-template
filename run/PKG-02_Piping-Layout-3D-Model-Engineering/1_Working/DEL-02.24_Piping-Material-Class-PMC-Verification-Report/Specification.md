# Specification: Piping Material Class (PMC) Verification Report

## 1. Scope

### 1.1 Purpose
This specification defines the requirements for developing the PMC Verification Report for the Puget Sound Optimization Project at Sumas Tank Farm.

### 1.2 Applicability
- All existing TM piping material classes used on the project
- Any new piping material classes developed for the project
- All codes and standards applicable to the project piping

### 1.3 Exclusions
- Vendor package internal piping (vendor responsibility)
- Instrument tubing material classes (Instrumentation scope)

## 2. Requirements

### 2.1 Verification Requirements

| Requirement | Description |
|-------------|-------------|
| REQ-01 | Verify PMC pressure-temperature ratings per ASME B31.3 / CSA Z662 |
| REQ-02 | Verify material specifications meet code requirements |
| REQ-03 | Verify flange ratings per ASME B16.5 / B16.47 |
| REQ-04 | Verify valve specifications per ASME B16.34 |
| REQ-05 | Verify fitting specifications per applicable standards |
| REQ-06 | Verify gasket specifications for service compatibility |
| REQ-07 | Verify bolt specifications per ASME B16.5 |
| REQ-08 | Identify any discrepancies from latest code edition |

### 2.2 Report Content Requirements

| Requirement | Description |
|-------------|-------------|
| REQ-09 | Include executive summary with key findings |
| REQ-10 | Document verification methodology |
| REQ-11 | List all codes and standards used with editions |
| REQ-12 | Provide detailed verification results per PMC |
| REQ-13 | Include discrepancy log with severity classification |
| REQ-14 | Provide resolution recommendations for all discrepancies |
| REQ-15 | Include supporting calculations as appendices |

## 4. Verification

### 4.1 Verification Methods

| Method | Description |
|--------|-------------|
| Code Comparison | Compare PMC ratings to code tables |
| Calculation Check | Verify pressure ratings calculations |
| Material Verification | Confirm material specs meet code |
| Peer Review | Independent review of verification |

### 4.2 Acceptance Criteria
- All PMCs verified against latest applicable codes
- All discrepancies documented with resolution recommendations
- Verification calculations traceable
- Owner review and approval of report
- Owner approval of discrepancy resolutions
