# Procedure: Piping Material Class (PMC) Verification Report

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for creating, reviewing, and issuing the PMC Verification Report for the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to all piping material classes used within the PSO project scope at Sumas Tank Farm.

## 2. Prerequisites

### 2.1 Input Documents

| Document | Source | Status Required |
|----------|--------|-----------------|
| TM PMC Sheets | TM Standard | Current issue |
| Design Basis | DEL-01.01 | Approved |
| Process Data | DEL-01.08 | Approved |
| Line List | DEL-02.01 | Draft minimum |
| Applicable Codes | Standards library | Latest editions |

### 2.2 Resources Required
- Piping Materials Engineer (Lead)
- Piping Engineer (support)
- Access to code library (ASME, CSA, ASTM)

### 2.3 Training Requirements
- Thorough knowledge of ASME B31.3 and CSA Z662
- Familiarity with piping component standards (B16.x series)
- Experience with material selection
