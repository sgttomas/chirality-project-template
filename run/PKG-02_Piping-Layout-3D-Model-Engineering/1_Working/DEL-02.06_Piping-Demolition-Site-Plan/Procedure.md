## 1. Purpose
This procedure defines the step-by-step process for developing the Piping Demolition Site Plan drawing(s) for the Puget Sound Optimization Project (PSO) at Sumas Tank Farm.

## 2. Scope
This procedure applies to the development, review, and issuance of piping demolition site plan drawings from FEED through Detailed Design phases.

## 3. Responsibilities

| Role | Responsibilities |
|------|-----------------|
| Piping Lead | Overall deliverable ownership, technical review, approval |
| Piping Designer/Engineer | Drawing development, coordination, revisions |
| CAD Technician | Drawing production per standards |
| Process Engineer | Confirmation of demolition scope from P&IDs |
| Civil/Structural Lead | Coordination of structural demolition scope |
| Document Controller | Document numbering, revision control, issuance |

## 4. Prerequisites

Before commencing drawing development:

|--------------|--------|--------|

## 5. Procedure

### Step 1: Gather Input Data
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 1.1 | Obtain existing condition documentation (as-builts, laser scan data) from TM/document control |
| 1.2 | Obtain current P&IDs (DEL-01.03) for scope identification |
| 1.3 | Review new design layout (DEL-02.05) to understand demolition requirements |
| 1.4 | Obtain draft Tie-In List (DEL-02.03) for cut point identification |
| 1.5 | Obtain TM CAD standards and drawing templates |
| 1.6 | Document any gaps in input data and request from TM |

**Output:** Collected input data package, gap list (if applicable)

---

### Step 2: Identify Demolition Scope
**Responsible:** Piping Designer/Engineer, with Process Engineer input

| Action | Details |
|--------|---------|
| 2.1 | Review P&IDs to identify existing piping within project footprint |
| 2.2 | Compare existing layout with new design to identify conflicts |
| 2.3 | Identify piping to be removed (mark on working drawings) |
| 2.4 | Identify cut points for tie-ins (coordinate with DEL-02.03) |
| 2.5 | Identify valves and inline equipment for removal |
| 2.6 | Identify piping to remain and preservation requirements |
| 2.7 | Coordinate with civil/structural for structural demolition scope |
| 2.8 | Document assumptions for any items not clearly defined |

**Output:** Demolition scope markup, assumptions register entries

---

### Step 3: Field Verification (If Required)
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 3.1 | Schedule site visit with TM coordination (per SOW requirements) |
| 3.2 | Complete JHA per DEL-00.12 requirements prior to site work |
| 3.3 | Verify existing conditions against as-built documentation |
| 3.4 | Document discrepancies with photographs and field notes |
| 3.5 | Update demolition scope based on field verification |

**Output:** Field verification record, updated scope documentation

---

### Step 4: Prepare Drawing Layout
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 4.1 | Set up drawing file per TM CAD standards |
| 4.2 | Import base survey/site plan data |
| 4.3 | Determine sheet coverage and create key plan |
| 4.4 | Establish match line locations (for multi-sheet sets) |
| 4.5 | Set up layers per demolition drawing requirements |
| 4.6 | Prepare title block with preliminary document number |

**Output:** Drawing template(s) ready for content

---

### Step 5: Develop Drawing Content
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 5.1 | Draft piping to be demolished with appropriate symbology |
| 5.2 | Add valve and equipment tags for items to be removed |
| 5.3 | Mark cut points and tie-in locations (reference DEL-02.03) |
| 5.4 | Add notation for items to remain ("EXIST TO REMAIN") |
| 5.5 | Add preservation/protection notes as applicable |
| 5.6 | Include reference callouts to related drawings (P&IDs, site plans) |
| 5.7 | Complete legend with all symbols used |
| 5.8 | Add general notes per specification requirements |
| 5.9 | Complete key plan showing sheet coverage |

**Output:** Draft drawing(s)

---

### Step 6: Internal Quality Check
**Responsible:** Piping Lead, Piping Designer

| Action | Details |
|--------|---------|
| 6.1 | Self-check against Specification requirements |
| 6.2 | Verify all demolished items traceable to P&IDs |
| 6.3 | Verify cut points align with Tie-In List |
| 6.4 | Check CAD standards compliance |
| 6.5 | Discipline check by Piping Lead |
| 6.6 | Inter-discipline coordination check (Process, C/S, Mechanical) |
| 6.7 | Document and resolve check comments |

**Output:** Checked drawings, resolved comment log

---

### Step 7: Issue for Review (IFR)
**Responsible:** Piping Lead, Document Controller

| Action | Details |
|--------|---------|
| 7.1 | Finalize drawings for IFR issue |
| 7.2 | Assign official document number per DEL-00.02 |
| 7.3 | Complete transmittal documentation |
| 7.4 | Submit to TM for review per SOW requirements |
| 7.5 | Log issue in MDR (DEL-00.01) |

**Output:** IFR drawings issued, transmittal record

---

### Step 8: Comment Resolution
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 8.1 | Receive and log TM review comments |
| 8.2 | Review and categorize comments |
| 8.3 | Prepare responses/resolutions for each comment |
| 8.4 | Coordinate clarifications with TM as needed |
| 8.5 | Implement accepted changes in drawings |
| 8.6 | Document comment resolution in log |

**Output:** Comment response log, updated drawings

---

### Step 9: Issue for Design (IFD)
**Responsible:** Piping Lead, Document Controller

| Action | Details |
|--------|---------|
| 9.1 | Incorporate all resolved comments |
| 9.2 | Final quality check |
| 9.3 | Update revision block and issue status |
| 9.4 | Issue IFD drawings |
| 9.5 | Update MDR |

**Output:** IFD drawings issued

---

### Step 10: Detailed Design Updates
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 10.1 | Review and incorporate vendor data impacts |
| 10.2 | Incorporate HAZOP action item resolutions |
| 10.3 | Update for design changes per change control |
| 10.4 | Coordinate continued updates with related deliverables |
| 10.5 | Progress to IFDE (Issued for Demolition Execution) |
| 10.6 | Progress to IFC as required |

**Output:** IFDE/IFC drawings

---

### Step 11: Native File Submission
**Responsible:** CAD Technician, Document Controller

| Action | Details |
|--------|---------|
| 11.1 | Prepare native CAD files per SOW-0259 requirements |
| 11.2 | Verify file naming convention compliance |
| 11.3 | Submit native files with PDF packages |
| 11.4 | Confirm receipt by TM document control |

**Output:** Native files submitted
