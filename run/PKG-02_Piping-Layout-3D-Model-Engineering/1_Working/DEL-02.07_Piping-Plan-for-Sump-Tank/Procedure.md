## 1. Purpose
This procedure defines the step-by-step process for developing the Piping Plan for Sump Tank drawing(s) for the Puget Sound Optimization Project (PSO) at Sumas Tank Farm.

## 2. Scope
This procedure applies to the development, review, and issuance of sump tank piping plan drawings from FEED through Detailed Design phases.

## 3. Responsibilities

| Role | Responsibilities |
|------|-----------------|
| Piping Lead | Overall deliverable ownership, technical review, approval |
| Piping Designer/Engineer | Drawing development, coordination, revisions |
| CAD Technician | Drawing production per standards |
| Process Engineer | Process requirements and system definition |
| Mechanical Engineer | Equipment data and coordination |
| I&C Engineer | Instrumentation requirements |
| Document Controller | Document numbering, revision control, issuance |

## 4. Prerequisites

Before commencing drawing development:

## 5. Procedure

### Step 1: Gather Input Data
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 1.1 | Obtain sump tank system P&ID from Process (DEL-01.03) |
| 1.2 | Obtain preliminary process datasheets for sump tank and pumps (DEL-01.09) |
| 1.3 | Obtain preliminary equipment data (size, nozzles) from Mechanical (DEL-03.01, DEL-03.02) |
| 1.4 | Review containment area layout for drain connection locations (DEL-04.21) |
| 1.5 | Obtain overall site layout context (DEL-02.05) |
| 1.6 | Obtain TM CAD standards and drawing templates |
| 1.7 | Identify and document any missing input data |

**Output:** Collected input data package, gap list (if applicable)

---

### Step 2: Develop Preliminary Layout
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 2.1 | Identify optimal sump tank location based on containment drainage |
| 2.2 | Position reinjection pump(s) relative to tank |
| 2.3 | Route major drain piping from containment areas |
| 2.4 | Plan pump suction and discharge routing |
| 2.5 | Identify valve and instrument locations |
| 2.6 | Verify adequate maintenance access |
| 2.7 | Coordinate with structural for foundation locations |
| 2.8 | Develop preliminary layout sketch for coordination |

**Output:** Preliminary layout sketch

---

### Step 3: Discipline Coordination
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 3.1 | Review layout with Process for P&ID alignment |
| 3.2 | Coordinate with Mechanical for equipment data |
| 3.3 | Coordinate with Civil/Structural for foundations |
| 3.4 | Coordinate with I&C for instrumentation locations |
| 3.5 | Incorporate 3D model for spatial verification |
| 3.6 | Document coordination decisions and changes |

**Output:** Coordinated layout, meeting notes

---

### Step 4: Prepare Drawing Layout
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 4.1 | Set up drawing file per TM CAD standards |
| 4.2 | Import site reference data (grids, adjacent equipment) |
| 4.3 | Determine drawing scale and sheet requirements |
| 4.4 | Set up layers per piping drawing requirements |
| 4.5 | Prepare title block with preliminary document number |

**Output:** Drawing template(s) ready for content

---

### Step 5: Develop Drawing Content
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 5.1 | Draft sump tank with dimensions and nozzle locations |
| 5.2 | Add pump(s) with proper orientation |
| 5.3 | Route all piping systems per layout |
| 5.4 | Add all valves with tag numbers |
| 5.5 | Show instrument connections and locations |
| 5.6 | Add pipe support location references |
| 5.7 | Include sections/elevations as needed for clarity |
| 5.8 | Add dimensions for equipment and piping |
| 5.9 | Complete legend with all symbols used |
| 5.10 | Add general notes and references |

**Output:** Draft drawing(s)

---

### Step 6: 3D Model Integration
**Responsible:** Piping Designer, 3D Modeler

| Action | Details |
|--------|---------|
| 6.1 | Model sump tank system in 3D (DEL-02.28) |
| 6.2 | Perform clash detection |
| 6.3 | Verify clearances and access |
| 6.4 | Update 2D drawings for any model-driven changes |
| 6.5 | Extract model views for drawing reference |

**Output:** 3D model complete, drawings updated

---

### Step 7: Internal Quality Check
**Responsible:** Piping Lead, Piping Designer

| Action | Details |
|--------|---------|
| 7.1 | Self-check against Specification requirements |
| 7.2 | Verify all P&ID items represented |
| 7.3 | Verify equipment data alignment |
| 7.4 | Check CAD standards compliance |
| 7.5 | Discipline check by Piping Lead |
| 7.6 | Inter-discipline coordination check |
| 7.7 | Document and resolve check comments |

**Output:** Checked drawings, resolved comment log

---

### Step 8: Issue for Review (IFR)
**Responsible:** Piping Lead, Document Controller

| Action | Details |
|--------|---------|
| 8.1 | Finalize drawings for IFR issue |
| 8.2 | Assign official document number per DEL-00.02 |
| 8.3 | Complete transmittal documentation |
| 8.4 | Submit to TM for review per SOW requirements |
| 8.5 | Log issue in MDR (DEL-00.01) |

**Output:** IFR drawings issued, transmittal record

---

### Step 9: Comment Resolution
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 9.1 | Receive and log TM review comments |
| 9.2 | Review and categorize comments |
| 9.3 | Prepare responses/resolutions for each comment |
| 9.4 | Coordinate clarifications with TM as needed |
| 9.5 | Implement accepted changes in drawings |
| 9.6 | Document comment resolution in log |

**Output:** Comment response log, updated drawings

---

### Step 10: Issue for Design (IFD)
**Responsible:** Piping Lead, Document Controller

| Action | Details |
|--------|---------|
| 10.1 | Incorporate all resolved comments |
| 10.2 | Final quality check |
| 10.3 | Update revision block and issue status |
| 10.4 | Issue IFD drawings |
| 10.5 | Update MDR |

**Output:** IFD drawings issued

---

### Step 11: Detailed Design Updates
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 11.1 | Incorporate vendor data (tank, pumps) |
| 11.2 | Incorporate HAZOP action item resolutions |
| 11.3 | Update for design changes per change control |
| 11.4 | Finalize dimensions and details |
| 11.5 | Coordinate with isometric development (DEL-02.13) |
| 11.6 | Progress to IFC |

**Output:** IFC drawings

---

### Step 12: Native File Submission
**Responsible:** CAD Technician, Document Controller

| Action | Details |
|--------|---------|
| 12.1 | Prepare native CAD files per SOW-0259 requirements |
| 12.2 | Verify file naming convention compliance |
| 12.3 | Submit native files with PDF packages |
| 12.4 | Confirm receipt by TM document control |

**Output:** Native files submitted
