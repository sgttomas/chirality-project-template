## 1. Purpose
This procedure defines the step-by-step process for developing the Overall Plot Plan drawing(s) for the Puget Sound Optimization Project (PSO) at Sumas Tank Farm.

## 2. Scope
This procedure applies to the development, review, and issuance of overall plot plan drawings from FEED through Detailed Design phases.

## 3. Responsibilities

| Role | Responsibilities |
|------|-----------------|
| Piping Lead | Overall deliverable ownership, technical review, approval |
| Piping Designer/Engineer | Drawing development, coordination, revisions |
| CAD Technician | Drawing production per standards |
| Civil Lead | Roads, grading, drainage input |
| Structural Lead | Building and foundation locations |
| Electrical Lead | Electrical building, area classification input |
| Project Engineer | Scope verification, stakeholder coordination |
| Document Controller | Document numbering, revision control, issuance |

## 4. Prerequisites

Before commencing drawing development:

|--------------|--------|--------|

## 5. Procedure

### Step 1: Gather Input Data
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 1.1 | Obtain site survey/topographic data from TM (DEL-04.05) |
| 1.2 | Review Appendix A-1 (Proposed Layout) for baseline arrangement |
| 1.3 | Review Appendix A-2 (Site Plan) for existing conditions |
| 1.4 | Obtain TM CAD standards and drawing templates |
| 1.5 | Obtain preliminary 3D model extract for current layout |
| 1.6 | Compile project scope summary for completeness check |
| 1.7 | Identify and document any missing input data |

**Output:** Collected input data package, gap list (if applicable)

---

### Step 2: Define Drawing Framework
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 2.1 | Determine project extents and required coverage |
| 2.2 | Select appropriate scale (typically 1:500 or 1:1000) |
| 2.3 | Determine sheet size and orientation |
| 2.4 | Define drawing boundaries and match lines |
| 2.5 | Establish coordinate grid to be shown |
| 2.6 | Plan legend, notes, and reference locations |

**Output:** Drawing framework definition

---

### Step 3: Develop Base Drawing
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 3.1 | Set up drawing file per TM CAD standards |
| 3.2 | Import site survey base data |
| 3.3 | Establish coordinate grid and datum references |
| 3.4 | Import existing facilities from as-built data |
| 3.5 | Apply existing facility symbology (screened/grey) |
| 3.6 | Prepare title block with preliminary document number |
| 3.7 | Set up layer structure per plot plan requirements |

**Output:** Base drawing with existing facilities

---

### Step 4: Develop New Facilities Content
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 4.1 | Import/extract new facilities from 3D model |
| 4.2 | Add booster pump station footprint and major equipment |
| 4.3 | Add manifold area footprint |
| 4.4 | Add containment wall extents |
| 4.5 | Add electrical building footprint |
| 4.6 | Add major piping corridors (simplified representation) |
| 4.7 | Add sump tank system location |
| 4.8 | Add access roads (new and modified) |
| 4.9 | Delineate project battery limits |
| 4.10 | Apply new facility symbology (bold/highlighted) |

**Output:** Drawing with new facilities added

---

### Step 5: Add Reference Information
**Responsible:** CAD Technician, Piping Designer

| Action | Details |
|--------|---------|
| 5.1 | Add coordinate grid labels |
| 5.2 | Add north arrow |
| 5.3 | Add graphic scale bar |
| 5.4 | Add match lines to area plans (DEL-02.09) |
| 5.5 | Add drawing references and cross-references |
| 5.6 | Complete legend with all symbols used |
| 5.7 | Add general notes |
| 5.8 | Add key plan (if multi-sheet) |

**Output:** Draft plot plan complete

---

### Step 6: Discipline Coordination
**Responsible:** Piping Designer, Design Coordinator

| Action | Details |
|--------|---------|
| 6.1 | Issue draft for Civil review (roads, grading) |
| 6.2 | Issue draft for Structural review (buildings, foundations) |
| 6.3 | Issue draft for Electrical review (electrical building, area classification) |
| 6.4 | Issue draft for Process review (scope completeness) |
| 6.5 | Collect and consolidate coordination comments |
| 6.6 | Resolve conflicts through coordination meetings |
| 6.7 | Incorporate agreed changes |
| 6.8 | Verify consistency with 3D model |

**Output:** Coordinated draft plot plan

---

### Step 7: Internal Quality Check
**Responsible:** Piping Lead, Piping Designer

| Action | Details |
|--------|---------|
| 7.1 | Self-check against Specification requirements |
| 7.2 | Verify complete project scope representation |
| 7.3 | Verify existing/new differentiation clarity |
| 7.4 | Check CAD standards compliance |
| 7.5 | Verify all cross-references correct |
| 7.6 | Discipline check by Piping Lead |
| 7.7 | Document and resolve check comments |

**Output:** Checked drawing, resolved comment log

---

### Step 8: Issue for Review (IFR)
**Responsible:** Piping Lead, Document Controller

| Action | Details |
|--------|---------|
| 8.1 | Finalize drawing for IFR issue |
| 8.2 | Assign official document number per DEL-00.02 |
| 8.3 | Complete transmittal documentation |
| 8.4 | Submit to TM for review per SOW requirements |
| 8.5 | Log issue in MDR (DEL-00.01) |

**Output:** IFR drawing issued, transmittal record

---

### Step 9: Comment Resolution
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 9.1 | Receive and log TM review comments |
| 9.2 | Review and categorize comments |
| 9.3 | Prepare responses/resolutions for each comment |
| 9.4 | Coordinate clarifications with TM as needed |
| 9.5 | Implement accepted changes in drawing |
| 9.6 | Document comment resolution in log |

**Output:** Comment response log, updated drawing

---

### Step 10: Issue for Design (IFD)
**Responsible:** Piping Lead, Document Controller

| Action | Details |
|--------|---------|
| 10.1 | Incorporate all resolved comments |
| 10.2 | Final quality check |
| 10.3 | Update revision block and issue status |
| 10.4 | Issue IFD drawing |
| 10.5 | Update MDR |

**Output:** IFD drawing issued

---

### Step 11: Detailed Design Updates
**Responsible:** Piping Designer/Engineer

| Action | Details |
|--------|---------|
| 11.1 | Incorporate design changes per change control |
| 11.2 | Update for vendor data impacts |
| 11.3 | Incorporate hazardous area classification |
| 11.4 | Maintain consistency with area plans and 3D model |
| 11.5 | Progress to IFC |

**Output:** IFC drawing

---

### Step 12: Native File Submission
**Responsible:** CAD Technician, Document Controller

| Action | Details |
|--------|---------|
| 12.1 | Prepare native CAD files per SOW-0259 requirements |
| 12.2 | Verify file naming convention compliance |
| 12.3 | Submit native files with PDF packages |
| 12.4 | Confirm receipt by TM document control |

**Output:** Native files submitted
