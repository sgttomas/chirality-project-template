## 1. Purpose and Scope

### 1.1 Purpose
This specification defines the requirements for the Overall Plot Plan drawing(s) to be developed as part of the Puget Sound Optimization Project (PSO) at Sumas Tank Farm. The plot plan shall provide a comprehensive view of the project facilities and their relationship to existing infrastructure.

### 1.2 Scope
The Overall Plot Plan shall cover:
- Complete PSO project footprint at Sumas Tank Farm
- New booster pump station with five VS6 pumps
- New 5x6 manifold installation
- New containment wall and secondary containment
- New electrical building
- Major piping corridors and connections
- Access roads and emergency egress
- Interfaces with existing facilities

### 1.3 Exclusions
- Detailed equipment arrangements (covered in Area Plans DEL-02.09)
- Detailed piping routing (covered in Piping Site Plan DEL-02.05)
- Detailed civil grading (covered in Civil Site Plans DEL-04.06)

## 2. Technical Requirements

### 2.1 Drawing Content Requirements

#### 2.1.1 General Requirements
| Requirement | Specification |
|-------------|---------------|
| Drawing Size | TBD (per TM CAD standards, typically E-size or larger) |

| Revision Block | Per TM document control requirements |

#### 2.1.2 Existing Facilities (Context)
The drawing shall show:
- Tank TK-101 and TK-102 (location and extent)
- Existing pump stations and manifolds (if adjacent)
- Existing buildings within/near project area
- Existing piping corridors
- Existing roads and fencing
- Property/lease boundaries (if applicable)

#### 2.1.3 New Facilities
The drawing shall show:

1. **Booster Pump Station**
   - Station footprint and boundaries
   - Individual pump locations (2 large, 2 small, 1 medium)
   - VFD/motor locations
   - Local control/instrumentation enclosures

2. **Manifold Area**
   - 5x6 manifold footprint
   - Valve arrangement area
   - Actuator locations

3. **Containment**
   - Containment wall extents
   - Secondary containment boundaries
   - Sump tank location

4. **Electrical Building**
   - Building footprint
   - Entrance/egress locations

5. **Piping**
   - Major piping corridors (schematic representation)
   - Tie-in locations to existing systems
   - Pipeline crossings (if any)

6. **Infrastructure**
   - Access roads (new and modified)
   - Emergency access routes
   - Fencing (new and modified)
   - Utility corridors

#### 2.1.4 Reference Information
- Site coordinate grid with labels
- Elevation reference datum
- Match lines to area plans (DEL-02.09)
- Key plan for multi-sheet sets
- Battery limits delineation
- Drawing references
