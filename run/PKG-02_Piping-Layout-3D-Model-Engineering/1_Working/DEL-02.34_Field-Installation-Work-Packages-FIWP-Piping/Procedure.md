# Procedure: Field Installation Work Packages (FIWP) - Piping

## 1. Purpose

### 1.1 Objective
This procedure defines the steps for developing, releasing, executing, and closing Field Installation Work Packages (FIWPs) for piping scope on the Puget Sound Optimization Project.

### 1.2 Scope
Applicable to all piping FIWPs within the PSO project scope at Sumas Tank Farm.
