# Specification: Field Installation Work Packages (FIWP) - Piping

## 1. Scope

### 1.1 Purpose
This specification defines the requirements for developing Field Installation Work Packages (FIWPs) for piping installation scope on the Puget Sound Optimization Project.

### 1.2 Applicability
- All field piping installation activities
- Pipe fabrication (shop and field)
- Pipe support installation
- Final connections and testing preparation
- Associated accessories installation

### 1.3 Exclusions
- Vendor package internal piping
- Instrument tubing
- Work covered by other discipline FIWPs

## 2. Requirements

### 2.1 Content Requirements
Each FIWP shall include:

| Requirement | Description |
|-------------|-------------|
| REQ-01 | FIWP cover sheet with unique identifier and work scope |
| REQ-02 | Detailed work description for crew execution |
| REQ-03 | All applicable drawings for the work scope |
| REQ-04 | Complete bill of materials with kitting status |
| REQ-05 | Required tools and equipment list |
| REQ-06 | Scaffold/access requirements |
| REQ-07 | Permit requirements (hot work, confined space, etc.) |
| REQ-08 | Quality requirements (ITP, hold points) |
| REQ-09 | Safety requirements (JHA, specific hazards) |
| REQ-10 | Weld list with WPS references |
| REQ-11 | Labor estimate and crew composition |

### 2.2 Format Requirements
- Package sized for crew-level execution (typically 1-2 weeks work)
- All drawings printed at readable scale
- Material list showing kitted/staged status
- Permit forms included (blank or prepared)
- Clear identification of hold points

### 2.3 Work Readiness Requirements
Before FIWP release, the following must be confirmed:
- Materials kitted and staged
- Scaffold/access available
- Permits obtained or obtainable
- Predecessor work complete
- Tools and equipment available
- Crew assigned

### 2.4 Quality Requirements
- Inspection and Test Plan (ITP) included
- Hold points clearly identified
- QC notification requirements documented
- Weld procedures referenced

## 4. Verification

### 4.1 Verification Methods
| Method | Description |
|--------|-------------|
| Constraint Check | All constraints cleared before release |
| Material Verification | Materials kitted and verified |
| Drawing Check | All required drawings included |
| Coordination | No conflicts with other FIWPs |

### 4.2 Acceptance Criteria
- 100% materials available and kitted
- All predecessor work complete
- All permits obtainable
- Crew assignment confirmed
- Foreman walkdown completed
