## 3. Scope Definition

### 3.1 Included Scope
- Location plans showing placement of all pipe supports within PSO scope
- Support specifications/callouts for each support location
- Coordinate references for support positioning
- Coverage of new booster pump station (5 x VS6-type pumps)
- Coverage of new 5x6 manifold system
- Coverage of associated infrastructure at SW corner of TK-102
- Integration with sitewide 3D model (DEL-02.28)

### 3.2 Excluded Scope
- Standard support detail drawings (covered by DEL-02.16)
- Existing supports outside PSO scope modifications
- Laurel Pump Station scope (explicitly excluded per SOW)
